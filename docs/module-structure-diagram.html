第一版核心活动流程

从账户进入到日常记录，再回到仪表盘查看评分。

注册 / 登录 手机号和密码进入系统，服务端维持登录态。
生成个性化计划 录入身体信息和健身目标，生成每日营养、饮水和睡眠目标。
记录饮食 按食物分类选择条目，输入克数，形成今日营养汇总。
记录训练与打卡 从动作库保存训练，运动打卡自动完成；饮水和睡眠手动输入。
查看仪表盘 汇总身体指标、营养进度、健康评分和近 7 日趋势。
